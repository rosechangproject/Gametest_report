題目二 — 📊 Jira Bug 數據自動化統計工具

練習目的： 解決手動統計 Bug 的耗時問題，透過 API 自動化抓取數據並產出視覺化報表與 Excel。

練習目的： 運用 Python 腳本自動化處理原始 API 資料，將複雜的欄位資訊轉化為易於閱讀的格式，並透過雲端同步提升資料管理效率。
